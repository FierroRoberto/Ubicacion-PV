# 📍 Registro de Ubicación de Clientes con selección de Excel

Aplicación web **responsive para móviles** que permite:
- Al iniciar, elegir entre **cargar un archivo Excel existente** o **crear uno nuevo con datos de ejemplo**.
- Seleccionar un cliente desde la lista extraída del Excel.
- Elegir frecuencia de visita (Semanal / Quincenal).
- Capturar ubicación GPS actual (latitud,longitud).
- Guardar los datos en el archivo Excel, actualizando la fila correspondiente.

## 🚀 Tecnologías usadas
- **Backend**: Node.js + Express + Multer (para subir archivos)
- **Manejo de Excel**: SheetJS (xlsx)
- **Frontend**: HTML5, CSS3, JavaScript vanilla (Geolocation API)
